# Codex Agent Instructions

## Project Overview
This repository is used to experiment with AI-assisted coding using Codex.

The goal is to build clean, well-structured code that is easy to understand and maintain.

---

## Tech Stack

Language:
- Python 3.10+

Framework:
- FastAPI (for web APIs)

Dependencies:
- pip
- requirements.txt

---

## Coding Rules

General rules:
- Follow PEP8 coding style
- Write clean and readable code
- Use meaningful variable names
- Add comments for complex logic

Functions:
- Keep functions short
- Each function should do one task

Error handling:
- Always handle possible exceptions

---

## Project Structure

Use this structure when creating files:

project/
│
├ README.md
├ AGENTS.md
├ requirements.txt
│
├ src/
│   └ main.py
│
└ tests/

---

## How to Run the Project

Run the application:

python src/main.py

---

## Testing

When possible:

- Add unit tests
- Use pytest

Example:

pytest

---

## Documentation

Always update:

- README.md
- code comments

when new features are added.

## Important Notes for Codex

When modifying code:

- Do not break existing functionality
- Prefer small incremental changes
- Explain major changes in comments
